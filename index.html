my-web-front

A lightweight (~2 KB) JavaScript utility library for everyday frontend tasks: DOM queries, element creation, events, delegation, debounce & throttle, and more.

~2 KB min Zero dependencies UMD + ESM MIT License

Quick Start

npm install my-web-front
or use a CDN: <script src="src/my-web-front.js"></script>

Why?

Modern browsers cover most needs, but common patterns like event delegation, debounce, and element creation still require boilerplate. my-web-front wraps these into a tiny, dependency-free toolkit exposed as mwf globally or as an ES module.

API Reference

mwf.qs(selector, ctx?)

Returns the first matching element (like document.querySelector).

const btn = mwf.qs('.submit-btn');
const inner = mwf.qs('.title', card); // scoped query

mwf.qsa(selector, ctx?)

Returns all matching elements as a real Array.

mwf.qsa('.item').forEach(el => el.style.color = 'red');

mwf.el(tag, attrs?, ...children)

Create a DOM element in a single expression.

const card = mwf.el('div', { class: 'card' },
  mwf.el('h3', null, 'Title'),
  mwf.el('p', null, 'Description text'),
  mwf.el('button', { onClick: handleClick }, 'Go')
);
document.body.appendChild(card);

mwf.on(node, event, handler, opts?)

Attach an event listener. Returns a detach function.

const off = mwf.on(btn, 'click', () => alert('hi'));
off(); // removes listener

mwf.once(node, event, handler)

Listen for an event only once.

mwf.delegate(parent, event, selector, handler)

Event delegation ‚Äî attach one listener, match children dynamically.

mwf.delegate(list, 'click', '.item', (e, item) => {
  console.log('Clicked:', item.textContent);
});

mwf.debounce(fn, ms)

Returns a debounced version of fn that waits ms after the last call.

const search = mwf.debounce(query => fetch(`/api?q=${query}`), 300);
input.addEventListener('input', e => search(e.target.value));

mwf.throttle(fn, ms)

Returns a throttled version of fn that fires at most once per ms.

mwf.addClass(node, ...classes)

Add one or more CSS classes to an element.

mwf.removeClass(node, ...classes)

Remove one or more CSS classes from an element.

mwf.toggleClass(node, cls, force?)

Toggle a class. Optional force boolean to explicitly add/remove.

mwf.hasClass(node, cls)

Check if an element has a given class.

mwf.ready(fn)

Run a function when the DOM is ready.

mwf.ready(() => {
  console.log('DOM loaded');
});

mwf.cssVar(name, value?)

Get or set a CSS custom property on :root.

mwf.cssVar('accent', '#ff6600'); // sets --accent
const c = mwf.cssVar('accent');      // reads --accent

mwf.fetchJSON(url, opts?)

Fetch wrapper that returns parsed JSON and throws on non-OK status.

mwf.fetchJSON('/api/data')
  .then(data => console.log(data))
  .catch(err => console.error(err));

Browser Support

All modern browsers (Chrome, Firefox, Safari, Edge). No polyfills needed.
